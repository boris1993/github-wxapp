## 搭建GitHub反向代理

你可以在LeanCloud上面创建一个免费应用，然后把 [这个项目](https://github.com/boris1993/github-api-reverse-proxy) 部署上去，
然后绑定一个已经备案过的域名，这样你就有一个`api.github.com`的反向代理了。
